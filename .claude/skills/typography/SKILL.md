# Typography Skill

# Purpose

You are responsible for creating a typography system that feels luxurious, modern, readable and consistent across the entire project.

Typography is the strongest visual element after whitespace.

Elegant typography creates perceived value.

Poor typography destroys trust.

---

# Typography Philosophy

Typography must feel:

Premium

Calm

Confident

Minimal

Timeless

Professional

Readable

Never decorative.

Never playful.

Never noisy.

---

# Primary Goal

Every visitor should instantly know:

What is important

What is secondary

What is clickable

What should be ignored

Typography creates hierarchy before color does.

---

# Font Selection Rules

Prefer clean modern sans-serif fonts.

Fonts should have:

Excellent readability

Professional appearance

Balanced proportions

Good Persian and Latin support when needed

Avoid novelty fonts.

Avoid script fonts.

Avoid decorative fonts.

Never mix unrelated font families.

---

# Font Weight System

Use only a limited set of weights.

Light → Large display text only.

Regular → Body text.

Medium → UI labels.

SemiBold → Buttons and cards.

Bold → Headlines.

Avoid excessive bold text.

---

# Heading Hierarchy

H1

One per page.

Largest visual element.

Clear and memorable.

---

H2

Major sections.

Strong hierarchy.

---

H3

Subsections.

---

H4

Component titles.

---

H5

Small titles when necessary.

Never skip heading levels without reason.

---

# Body Text

Readable at all sizes.

Comfortable line spacing.

Never compress paragraphs.

Avoid long uninterrupted text blocks.

Keep paragraphs short.

---

# Line Length

Ideal reading width:

Approximately 60–80 characters.

Avoid extremely wide text blocks.

---

# Line Height

Generous line spacing.

Reading should feel effortless.

Large headings may use tighter spacing.

Body text should breathe.

---

# Letter Spacing

Default letter spacing should remain natural.

Only large display headings may receive subtle adjustments.

Never overuse tracking.

---

# Alignment

Left-aligned for Latin.

Right-aligned for Persian.

Center alignment only for:

Hero headings

Short marketing statements

Never center long paragraphs.

---

# Hero Typography

Large.

Confident.

Minimal.

Strong hierarchy.

Short sentences.

Avoid long marketing headlines.

---

# Product Typography

Product name

Most important.

Price

Second priority.

Specifications

Third priority.

Description

Readable but visually lighter.

---

# Button Text

Short.

Clear.

Action-oriented.

Examples:

Buy Now

View Collection

Learn More

Add to Cart

Avoid vague labels.

---

# Navigation

Navigation typography should be simple.

Medium weight.

Comfortable spacing.

Never crowded.

---

# Forms

Labels should always remain visible.

Placeholder text should never replace labels.

Error messages should be clear.

Helper text should be subtle.

---

# Cards

Title

Primary.

Description

Secondary.

Metadata

Muted.

Action

Obvious.

---

# Numbers

Prices should stand out naturally.

Do not exaggerate size.

Weight should communicate importance.

Align numbers consistently.

---

# Luxury Style

Luxury brands rarely shout.

Avoid:

ALL CAPS paragraphs

Multiple exclamation marks

Large bold text everywhere

Aggressive typography

Use confidence instead of volume.

---

# Readability

Contrast first.

Spacing second.

Size third.

Decoration last.

Readable typography always wins.

---

# Responsive Typography

Typography should scale smoothly.

Desktop

Tablet

Mobile

Maintain hierarchy on every device.

Never allow headings to dominate small screens.

---

# Accessibility

Readable contrast.

Scalable text.

Proper heading order.

Semantic HTML.

Screen reader friendly.

Never communicate meaning by size alone.

---

# Emphasis Rules

Use emphasis sparingly.

Prefer:

Weight

Size

Spacing

Before using color.

Bold should mean something important.

---

# Quotes

Quotes should feel editorial.

Generous spacing.

Distinct but subtle styling.

Never oversized quotation marks.

---

# Lists

Keep lists clean.

Consistent spacing.

Easy scanning.

Never overcrowd bullet points.

---

# Empty States

Typography should guide users.

Explain:

What happened.

Why.

What to do next.

Avoid robotic wording.

---

# Error Messages

Clear.

Human.

Actionable.

Never blame the user.

---

# Microcopy

Every sentence should reduce friction.

Use friendly professional language.

Avoid unnecessary technical wording.

---

# Things To Avoid

Never use:

Tiny text

Huge paragraphs

Decorative fonts

Random font sizes

Inconsistent weights

Multiple font families

Crowded text

Overly bold interfaces

---

# Final Checklist

Before approving any UI ask:

Is hierarchy obvious?

Can users scan the page easily?

Is reading comfortable?

Is typography consistent?

Does the page feel premium?

Could any text be simplified?

Typography should make the interface feel calm, expensive and trustworthy.

That is the standard for every page.
