# Animation & Motion Skill

# Purpose

You are responsible for every animation, transition and micro-interaction in this project.

Animation is not decoration.

Animation explains.

Animation guides.

Animation gives confidence.

Animation should never become the center of attention.

---

# Philosophy

Luxury products move with confidence.

The interface should feel:

Elegant

Natural

Smooth

Calm

Responsive

Intentional

Every movement should have a reason.

---

# Golden Rule

If removing an animation improves the experience...

Remove it.

Motion exists to improve usability.

Never animate for entertainment.

---

# Motion Principles

Every animation must satisfy at least one of these:

Guide attention

Explain change

Provide feedback

Reduce cognitive load

Increase perceived quality

If none apply, do not animate.

---

# Animation Speed

Very Fast

100–150ms

Standard

180–250ms

Large Layout Changes

250–350ms

Never exceed 500ms for interface transitions.

Luxury feels responsive.

Not slow.

---

# Easing

Use natural easing.

Acceleration should feel soft.

Stopping should feel gentle.

Avoid linear animations.

Avoid elastic or bouncing effects.

---

# Hover Effects

Hover effects should be subtle.

Examples:

Slight elevation

Soft shadow

Small background change

Tiny scale (1.01–1.02)

Gentle border transition

Never create dramatic hover animations.

---

# Buttons

Buttons should respond immediately.

Hover

Soft elevation

Press

Slight compression

Loading

Smooth state transition

Disabled

Reduced opacity

Never use flashy button effects.

---

# Cards

Hover should indicate interactivity.

Allowed:

Small lift

Soft shadow

Border refinement

Image zoom (very subtle)

Forbidden:

Large rotations

Aggressive scaling

3D flips

---

# Images

Images may:

Fade in

Zoom slightly on hover

Reveal gently

Never spin.

Never bounce.

Never shake.

---

# Navigation

Menus should:

Fade

Slide softly

Appear instantly

Never drop dramatically.

Never overshoot.

---

# Dropdowns

Fast.

Elegant.

No bouncing.

No exaggerated scaling.

Maintain spatial continuity.

---

# Modals

Fade

-

Small scale

Background should dim softly.

Closing animation should feel symmetrical.

---

# Page Transitions

Keep page transitions minimal.

Fast fade.

Soft movement.

Content should remain stable.

Avoid long transition screens.

---

# Scroll Animations

Allowed:

Fade up

Fade in

Soft reveal

Small translate

Forbidden:

Flying elements

Rotating sections

Random directions

Heavy parallax

---

# Hero Sections

Hero content may appear sequentially.

Headline

↓

Description

↓

CTA

↓

Image

Timing should feel effortless.

---

# Lists

Reveal progressively.

Keep stagger intervals small.

Never delay usability.

---

# Product Cards

Hover

Image zoom slightly.

Shadow softens.

CTA becomes clearer.

Nothing else.

---

# Loading

Prefer skeleton loaders.

Avoid spinning loaders whenever possible.

Loading should reassure users.

Not distract them.

---

# Success Feedback

Short.

Subtle.

Positive.

Examples:

Checkmark fade

Button confirmation

Toast slide

Never celebrate excessively.

---

# Error Feedback

Gentle shake (small)

or

Color transition

Never aggressive shaking.

Never flashing.

---

# Notifications

Appear softly.

Disappear automatically.

Remain readable.

Do not interrupt the workflow.

---

# Scroll Behavior

Scrolling should feel smooth.

Sticky elements should move naturally.

Avoid scroll-jacking.

Never hijack user control.

---

# Micro Interactions

Every interaction should acknowledge the user.

Click

Hover

Focus

Selection

Toggle

Accordion

Tabs

Each should provide subtle visual feedback.

---

# Performance

Prefer CSS transitions.

Avoid expensive JavaScript animations.

Use GPU-friendly properties.

Animate:

Opacity

Transform

Avoid animating:

Width

Height

Top

Left

Large shadows

Blur

---

# Accessibility

Respect reduced motion preferences.

If reduced motion is enabled:

Disable unnecessary animations.

Maintain usability.

Never rely on animation alone to communicate meaning.

---

# Luxury Motion

Luxury motion is almost invisible.

Users should feel the quality.

Not notice the animation.

---

# Never Use

Bounce

Flash

Spin

Rubber-band

Random delays

Constant movement

Infinite decorative animations

Blinking elements

Confetti

Excessive particle effects

Over-animated pages

---

# Final Checklist

Before approving motion ask:

Does this animation improve usability?

Is it smooth?

Is it subtle?

Does it feel premium?

Would Apple, Stripe or Linear use this motion?

Can the interface work perfectly without it?

If the answer is yes, keep it.

If not, simplify it.

Goal:

Every interaction should feel refined, effortless and worthy of a premium luxury brand.
